Sorting algorithm

Bubble Sort

A simple comparison-based algorithm for learning how adjacent swaps gradually move values into place.

Bubble Sort repeatedly steps through the list, compares adjacent elements, and swaps them when they are out of order. It is one of the clearest ways to understand sorting mechanics, but it becomes impractical on larger inputs.

Step by Step

  • Start at the beginning of the array.
  • Compare each adjacent pair.
  • Swap if the left value is greater than the right.
  • Repeat until a full pass makes no swaps.

Usage Notes

Use it when

You want a simple visual demonstration or are working with a tiny dataset.

Avoid it when

Performance matters or the input size is large enough for O(n²) cost to dominate.

Explanation

How Bubble Sort Works

Bubble Sort compares each pair of adjacent elements and swaps them if they are in the wrong order. After each pass, the largest unsorted element "bubbles" to its correct position at the end.

In the average and worst cases, the algorithm performs on the order of \( \frac{n(n - 1)}{2} \) comparisons, which is why its running time grows as \( O(n^2) \).

What Makes It Different

Bubble Sort only works with adjacent swaps. That makes it easy to follow visually, but it also means small values can take many steps to move all the way to the front.

Pseudocode

for i from 0 to n - 1:
    for j from 0 to n - i - 1:
        if array[j] > array[j + 1]:
            swap(array[j], array[j + 1])
